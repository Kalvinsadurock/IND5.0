# First 90-Day Build Roadmap

## Days 1-30: Platform Foundation
- Tenant, plant, user, role, permission model
- Audit log
- Workflow engine foundation
- Form/configuration engine foundation
- Canonical master data model
- Industry template structure
- Module shells for MES, ERP, HRMS, OEE, Quality, Work Management

## Days 31-60: Core Industrial Workflows
- MES process definition and work order execution
- Inventory material and lot tracking
- Quality checkpoints and defect management
- HRMS employee, onboarding, skill, certification foundation
- Work board for production, quality, maintenance, integration tickets
- Basic reports and dashboards

## Days 61-90: Performance and Enterprise Integration
- OEE shifts, downtime, good/reject counts, dashboards
- ERP procurement and stock movement core
- SAP integration prototype and reconciliation model
- Workforce productivity and shift allocation
- Traceability reports
- Industry template pilot for one target industry

## MVP Priority Order
1. Platform Core
2. Master Data
3. Configuration Studio foundation
4. MES Execution
5. Inventory
6. Quality
7. HRMS Workforce
8. OEE
9. Work Management
10. Reports
11. SAP Integration prototype
