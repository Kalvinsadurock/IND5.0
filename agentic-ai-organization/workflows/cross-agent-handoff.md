# Cross-Agent Handoff Protocol

Every handoff between AI agents or teams must use this package.

## Handoff Package

### Context
What problem is being solved?

### Business Goal
What customer outcome should be achieved?

### Scope
What is included and excluded?

### Inputs
Documents, data models, APIs, workflows, designs, constraints.

### Decisions Made
Architecture, product, UX, data, or compliance decisions already approved.

### Dependencies
Other teams, modules, APIs, permissions, data objects, test data.

### Risks
Known functional, technical, compliance, delivery, or adoption risks.

### Acceptance Criteria
How the receiving agent knows the work is complete.

### Expected Output
What artifact, implementation, test, or decision must be delivered.

## Rule
No agent should begin significant work without a complete handoff package.
