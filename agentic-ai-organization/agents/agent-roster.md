# Agent Roster

## Standard Module Team

Each module team contains these agents:

| Agent | Responsibility |
|---|---|
| Module Solution Architect | Designs module architecture, APIs, data model, events, integrations, scalability. |
| Module Product Owner | Defines user value, requirements, personas, acceptance criteria, customer scenarios. |
| Module Product Manager | Prioritizes roadmap, packaging, market fit, editions, and release scope. |
| Business Analyst | Converts workflows into user stories, rules, forms, process maps, and edge cases. |
| UX/UI Agent | Designs role-based screens, dashboards, forms, navigation, and mobile/tablet flows. |
| Developer Agent | Builds frontend, backend, APIs, database objects, workflow logic, integrations. |
| QA Agent | Creates and executes functional, edge-case, regression, integration, and permission tests. |
| Reporting Agent | Defines dashboards, KPIs, exports, operational reports, and analytics models. |
| Compliance Agent | Validates audit trail, role access, e-signature, traceability, safety, and regulation needs. |

## Core Teams

| Team | Scope |
|---|---|
| Platform Core | Tenant, plant, roles, permissions, workflow engine, form builder, audit log, notifications. |
| Configuration Studio | No-code process, form, workflow, report, dashboard, terminology, and industry template configuration. |
| MES | Process flows, work orders, execution, traceability, digital traveler, operator workflows. |
| ERP and Inventory | BOM, routing baseline, procurement, stock, warehouse, costing, sales, dispatch. |
| SAP Integration | SAP-compatible mappings, sync logs, posting queues, reconciliation, integration error tickets. |
| OEE | Availability, performance, quality, downtime, speed loss, good/reject counts, shift dashboards. |
| HRMS and Workforce | Employees, onboarding, skills, certification, shifts, attendance, safety eligibility, labor productivity. |
| Quality and Compliance | Inspection plans, defects, NCR, CAPA, deviation, rework, scrap, audit evidence. |
| Maintenance and Asset | Machines, tools, moulds, assets, PM, breakdowns, calibration, spares, machine health. |
| Work Management | Tickets, sprint boards, CAPA tasks, maintenance actions, engineering changes, escalations. |
| Reporting, Analytics, and AI | Reports, dashboards, KPI layer, anomaly detection, recommendations, root-cause suggestions. |
| Security and Release | RBAC, tenant isolation, release train, environments, monitoring, backup, deployment quality. |
